# 人脸识别 + 表情识别程序 - 超简单使用指南

这个程序可以：
- 用摄像头实时识别人的脸
- 认识谁就显示谁的名字
- 不认识的人可以随时添加到系统里
- **识别人脸表情（开心、悲伤、愤怒等）**
- 整个系统可以复制到其他电脑使用
- 所有处理都在本地完成，保护隐私

---

## 第一步：安装（只需要做一次）

打开"命令提示符"（按 Win+R，输入 cmd，回车），然后复制粘贴下面这行命令并按回车：

```
pip install opencv-python torch numpy facenet-pytorch ultralytics deepface
```

等待安装完成（可能需要几分钟）。

---

## 第二步：启动程序

1. 进入 quickface 文件夹
2. 双击 `quick_face_recognition.py` 右键选择 "用 Python 打开"

或者在命令提示符中进入文件夹后输入：
```
python quick_face_recognition.py
```

启动后：
- 会弹出一个窗口显示摄像头画面
- 第一次运行会自动下载模型文件（需要联网，可能需要等几分钟）
- 程序准备好了会显示 "摄像头已打开，开始识别..."

---

## 第三步：如何使用

### 📷 看到摄像头画面后：

| 按键 | 功能 |
|------|------|
| **q** | 退出程序（关闭窗口或按 q） |
| **r** | 添加新面孔（输入名字，按回车确认） |
| **s** | 保存当前画面截图 |
| **e** | 开关表情识别功能 |

### 🎭 表情识别功能

程序可以识别 7 种表情：

| 表情 | 说明 |
|------|------|
| 😊 开心 | 快乐、微笑 |
| 😢 悲伤 | 不开心、难过 |
| 😠 愤怒 | 生气的表情 |
| 😨 恐惧 | 害怕、紧张 |
| 😲 惊讶 | 吃惊、意外 |
| 🤢 厌恶 | 不喜欢、反感 |
| 😐 平静 | 没有明显表情 |

### 画面显示说明：

```
张三                    ← 绿色文字：已识别的人
😊 开心 (85%)          ← 黄色文字：表情和置信度
┌─────────────────┐
│                 │
│    👤           │  ← 绿色边框：识别成功
│                 │
└─────────────────┘
```

如果显示 "Unknown" 和红色边框，表示这个人还没注册。

---

## 第四步：添加新面孔（注册人脸）

详细步骤：

1. 让要添加的人脸对准摄像头
2. 按 **r** 键（窗口下方会显示 "Register:"）
3. 用键盘输入这个人的名字（比如"张三"、"小明"）
4. 按 **回车键** 确认
5. 如果成功，命令窗口会显示 "已注册: 张三"

以后这个人再出现，程序就会显示他的名字和表情。

---

## 第五步：查看和管理已注册的人脸

程序会在同一个文件夹里自动创建两个东西：

```
quickface/
├── quick_face_recognition.py    ← 主程序
├── face_database.json            ← 人脸数据库（记住的人）
└── face_images/                  ← 保存的图片文件夹
```

- **face_database.json** - 里面存储了所有已注册的人脸信息
- **face_images/** - 里面是保存的截图

---

## 第六步：批量添加人脸（从照片）

如果你有一堆照片想要一次性添加进去：

### 方法 A：使用专用脚本（推荐）

1. 把照片放在一个文件夹里
2. 照片的文件名就是人的名字（比如：张三.jpg、小明.png）
3. 打开命令提示符，进入 quickface 文件夹
4. 输入：
   ```
   python register_from_images.py
   ```
5. 程序会问你要哪个文件夹，输入照片所在的文件夹路径
6. 等待处理完成

### 方法 B：手动添加

1. 按 **s** 键保存当前画面
2. 打开 `face_images` 文件夹，找到刚才保存的图片
3. 把图片重命名为你想注册的人名（比如改成 "老王.jpg"）
4. 运行批量注册脚本，指向 `face_images` 文件夹

---

## 如何把程序搬到另一台电脑

只需要把这些文件复制过去：

**必须复制：**
- ✅ `quick_face_recognition.py`（主程序）
- ✅ `face_database.json`（人脸数据库）

**可选复制：**
- 📁 `face_images/`（人脸图片，没有也不影响识别）

复制后，在新电脑上安装依赖（第一步的命令），然后运行程序就行！

## 表情识别说明

### 开启/关闭表情识别
- 按 **e** 键可以开关表情识别功能
- 关闭表情识别可以提高运行速度

### 表情准确度
- 光线要好，表情要清晰
- 正面朝向摄像头效果最好
- 置信度显示在百分比，数字越高越准确

### 表情识别速度
- 使用了缓存机制，不会每帧都分析
- 可以流畅运行

## 常见问题

### Q: 第一次运行很慢，卡住了？
A: 正常的，正在下载模型文件。保持联网，耐心等待几分钟。

### Q: 摄像头打不开？
A: 检查摄像头是否被其他程序占用了，关闭其他使用摄像头的软件再试。

### Q: 表情识别不准确？
A: 光线要好，脸要正对摄像头，不要侧脸或被遮挡，表情要清晰。

### Q: 想提高运行速度？
A: 按 `e` 键关闭表情识别，或者修改代码中的 `emotion_cache_ttl` 参数增加缓存时间。

### Q: 表情识别不工作？
A: 检查是否安装了 `deepface` 库，运行：
   ```
   pip install deepface
   ```

### Q: 想删除已注册的人？
A: 用记事本打开 `face_database.json`，删除对应的行，然后保存。

### Q: 识别阈值是什么？
A: 程序判断"是不是同一个人"的标准，数字越小要求越严格。代码默认是 1.2，如果经常认错人可以改小一点（比如 1.0）。

## 程序工作原理

### 人脸识别流程：
1. **YOLO** - 快速找到画面里的人脸在哪
2. **MTCNN** - 把找到的人脸对齐、调整大小
3. **FaceNet** - 提取人脸的"特征"（像指纹一样）
4. **比对** - 拿当前的人脸特征和数据库里的比对

### 表情识别流程：
1. **DeepFace** - 使用深度学习模型分析人脸表情
2. **分类** - 判断是开心、悲伤、愤怒等7种表情
3. **置信度** - 给出判断的可信程度
